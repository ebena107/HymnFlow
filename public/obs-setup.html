🎛️ Global OBS Hotkeys New in 2.5

Global hotkeys work from any window in OBS — you don't need to click the dock first. They use OBS's built-in hotkey system and communicate via obs-websocket (included in OBS 28+).

  1. Enable obs-websocket in OBS
    • Open OBS → Tools → obs-websocket Settings
    • Check Enable WebSocket server
    • Note the port (default: 4455) and set a password if desired
    • Click OK
  2. Load the HymnFlow hotkey script
    • Open OBS → Tools → Scripts
    • Click + and select scripts/hymnflow-obs-hotkeys.py
    • In the script panel that appears, enter your obs-websocket host, port, and password
  3. Bind your keys
    • Open OBS → Settings → Hotkeys
    • Search "HymnFlow" — you'll see 7 actions
    • Click the field next to each action and press your desired key
    • Click OK
  4. Connect HymnFlow dock to obs-websocket
    • In the HymnFlow dock, go to the Style tab → OBS Connection section
    • Enter the same host (127.0.0.1), port, and password
    • Click Connect — the badge turns green ●
    • HymnFlow will auto-reconnect on OBS restarts once connected

📅 Service Planning Workflow New in 2.5

  1. Create a service: Service tab → + New Service → give it a name (e.g., "Sunday Morning 22 Jun")
  2. Add items: Click + Hymn, + Scripture, + Announcement, or + Divider to build the order
  3. Reorder: Drag items up and down to arrange. Inline ↑/↓ buttons also available.
  4. Add notes: Each item has an optional internal notes/cue field (not shown on screen)
  5. Save the service. You can duplicate it as a template for next week.
  6. Load & run: Click Load on a service — the banner appears at the top of the dock showing the current item + "Next: …". Use ] or the banner → arrows to advance.

📖 Live Streaming Workflow

  1. Before Stream: Build your service order in the Service tab and import your hymn library
  2. Load Service: Click Load on your service — items appear in order in the banner
  3. When Hymn Starts: The dock auto-selects the hymn; press Space to display verse 1
  4. Navigate: → / ← between verses, ↓ / ↑ for line windows within long verses
  5. Next Item: Press ] (or the banner arrow) — or let the last verse auto-advance
  6. Scripture: Activating a scripture item switches to Live tab and loads the verse ready to display
  7. Done: Press Space to hide the overlay

💡 Tips

  • Lines per display: Adjust slider (1–6) to control how many lines show at once — great for long verse text
  • Preview: Watch the preview box in the Live tab to see exactly what will display before pressing Show
  • Backup: Export your hymns regularly (Library → Export) to avoid data loss
  • Performance: Use transparent background for the cleanest overlay quality on any scene
  • Language: Change interface language in the Style tab → Settings (9 languages)
  • Global hotkeys: Bind F5/F6 to Next/Prev Verse in OBS Settings → Hotkeys for hands-free control
  • Multiple Bibles: Load KJV + your local language translation — switch between them mid-service with the Live tab selector
  • Service templates: Use the Dup button to copy last week's service as a template for this week

❓ Troubleshooting

Dock doesn't appear

  • URL must use forward slashes: file:///C:/HymnFlow/…
  • Confirm the path is correct
  • Restart OBS after adding the dock

Overlay doesn't update

  • Both dock and overlay must use the same origin (same file:// folder or same HTTP server)
  • Right-click the Browser Source → Refresh
  • Check the source is in an active scene

Bible lost after OBS restart

  • v2.5 stores Bibles in IndexedDB — should persist reliably
  • If upgrading from v2.4.x, re-import your Bible once — it will migrate automatically

Global hotkeys not firing

  • Confirm obs-websocket is enabled: OBS → Tools → obs-websocket Settings
  • Confirm the script is loaded: OBS → Tools → Scripts
  • The dock badge should show green ● when connected
  • Keys must be bound in OBS Settings → Hotkeys (search "HymnFlow")
