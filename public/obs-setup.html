📖 Bible Lookup Setup

The Quick Scripture panel lets you look up and display verses live. Bible data is not bundled — import a free public domain JSON file.

Step 1: Get a Bible JSON File

Download the KJV (public domain) from one of these GitHub repositories:

  • thiagobodruk/bible → en_kjv.json — click Raw, then Save As → kjv.json
  • aruljohn/Bible-kjv → kjv.json — click Raw, then Save As → kjv.json

Or if you have Python 3: run python scripts/bundle_bible_kjv.py — it downloads and converts automatically.

Step 2: Import into HymnFlow

  1. Open HymnFlow Dock → Library tab → Bible section
  2. Type KJV in the name field
  3. Click Import JSON and select the file
  4. Status badge shows KJV ✓ — you're ready

Step 3: Use Quick Scripture

  1. Switch to the Live tab → Scripture panel
  2. Type a reference: John 3:16, Ps 23, 1 Cor 13:4-7, Jude 1
  3. Press 🔍 or Enter to look up
  4. Press Display to send to overlay, then use ← → to step through verses

📖 Usage Example

Live Streaming Workflow:

  1. Before Stream: Import your hymn library
  2. During Service: Keep dock open in OBS
  3. When Hymn Starts: Click hymn from list
  4. Show First Verse: Press Show or Space
  5. Navigate: Use arrow keys to move through verses/lines
  6. When Done: Press Hide

🌍 Changing Interface Language

HymnFlow supports 9 languages for the user interface:

  • 🇬🇧 English (en) - Default
  • 🇪🇸 Spanish (es - Español)
  • 🇫🇷 French (fr - Français)
  • 🇵🇹 Portuguese (pt - Português)
  • 🇹🇿 Swahili (sw - Kiswahili)
  • 🇵🇭 Tagalog (tl)
  • 🇳🇬 Yoruba (yo - Yorùbá)
  • 🇨🇳 Mandarin Chinese (zh - 中文)
  • 🇰🇷 Korean (ko - 한국어)

To change language:

  1. Open the HymnFlow Dock in OBS
  2. Click the Style tab
  3. Find the Interface Language dropdown
  4. Select your preferred language
  5. The UI updates immediately - all labels, buttons, and tooltips will be in your selected language
  6. Your preference is automatically saved in browser localStorage
🎉 You're all set! Start by clicking "Open Dock (Test)" above to try the interface.
